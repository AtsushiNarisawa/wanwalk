# TestFlight 説明文 v1.0.0+10

## 📝 日本語版（推奨）

### 新機能（What to Test）
```
【v1.0.0+10の新機能】

✨ UI/UXの大幅改善
• TOPページのカードデザインを一新
• グラデーション背景で視覚的に豊かな表示
• エリアごとに異なるカラーテーマ

🗺️ 箱根エリアの改善
• 箱根を5つのサブエリアに分割（箱根湯本・宮ノ下・強羅・仙石原・芦ノ湖）
• 各エリアへの交通アクセス情報を追加
• エリア選択がより直感的に

📱 TOPページの改善
• コンテンツの表示順序を最適化
• 最新のピン投稿を最上部に表示
• 箱根観光デジタルマップへのリンク追加

【テストポイント】
1. TOPページのカードデザインをご確認ください
2. 箱根エリアをタップして、サブエリア選択をお試しください
3. 各箱根サブエリアで交通情報が表示されることをご確認ください
4. おすすめエリアセクションのバナーをタップしてみてください

ご意見・ご感想をお待ちしております！
```

### テスト方法（Test Details）
```
【テスト手順】

1. アプリ起動
   → TOPページが表示されることを確認

2. TOPページの確認
   → セクション順序：①最新ピン投稿 ②人気コース ③おすすめエリア ④高評価スポット
   → カードにグラデーション背景が表示されているか確認

3. 箱根エリアのテスト
   → MAPタブ → 「箱根」カードをタップ
   → 5つのサブエリア（仙石原、芦ノ湖など）が表示されるか確認
   → サブエリアをタップ → 交通情報カードが表示されるか確認

4. バナーのテスト
   → TOPページ → おすすめエリアセクション
   → 「箱根観光デジタルマップ」バナーをタップ
   → 外部ブラウザが開くか確認

5. 既存機能の確認
   → ピン投稿の閲覧・いいね・コメント
   → ルート詳細の表示
   → 地図表示とGPS機能

【フィードバックのお願い】
• 使いやすさの改善点
• デザインの印象
• 不具合やエラー
• 新機能のご要望
```
