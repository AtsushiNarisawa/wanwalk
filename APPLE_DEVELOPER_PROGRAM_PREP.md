## 📝 概要

このドキュメントは、WanMap を Apple App Store に公開するための Apple Developer Program 申請準備を支援します。

### 3. App Store スクリーンショット（必須）

以下のデバイスサイズのスクリーンショットが必要です：

- **6.5インチ ディスプレイ** (iPhone 11 Pro Max, XS Max)
  - サイズ: 1242 x 2688 pixels

#### iPad 用（任意、でも推奨）
- **12.9インチ ディスプレイ** (iPad Pro)

### 8. プライバシーポリシー（必須文書）

以下の内容を含む必要があります：

```markdown
# WanMap プライバシーポリシー

最終更新日: 2024年11月14日

## 収集する情報

### 位置情報
- 散歩ルートの記録のため、GPS位置情報を使用します
- 位置情報は散歩記録中のみ取得され、バックグラウンドでは取得しません
- ユーザーの同意なしに第三者と共有することはありません

### 個人情報
- メールアドレス（アカウント作成時）
- ユーザー名とプロフィール情報
- 散歩記録（距離、時間、ルート情報）
- アップロードした写真

### 使用目的
- アプリの機能提供
- ユーザーサポート
- サービス改善

## データの保存

- データは Supabase（PostgreSQL）に保存されます
- 写真は Supabase Storage に保存されます
- データは暗号化されて保存されます

## データの共有

### 公開される情報
- ユーザー名とプロフィール写真
- 公開設定にした散歩ルート
- コメントとレビュー

### 非公開の情報
- メールアドレス
- 位置情報の生データ
- 非公開設定のルート

## ユーザーの権利

- いつでもアカウントを削除可能
- データのエクスポート要求が可能
- プライバシー設定の変更が可能

## お問い合わせ

プライバシーに関するご質問は以下までご連絡ください：
contact@doghub-hakone.com
```

## 🔧 技術的な準備

### 1. Info.plist の設定

位置情報の使用目的を明記：

```xml
<key>NSLocationWhenInUseUsageDescription</key>
<string>散歩ルートを記録するために位置情報を使用します</string>

<key>NSCameraUsageDescription</key>
<string>散歩中の写真を撮影するためにカメラを使用します</string>

<key>NSPhotoLibraryUsageDescription</key>
<string>写真を選択してルートに追加するためにフォトライブラリへのアクセスが必要です</string>
```

### 2. Build Settings

```
Bundle ID: com.doghub.wanmap
Version: 1.0.0
Build Number: 1
Deployment Target: iOS 15.0+
```
